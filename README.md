# ThugilMart

ThugilMart is a React + Vite e-commerce web app.

## Tech Stack

- React, Vite
- React Context API for state
- Vercel deployment

## Installation

```bash
git clone https://github.com/kanyacse/ThugilMart.git
cd ThugilMart
npm install
npm run dev

Project URL: https://thugilmart.vercel.app
